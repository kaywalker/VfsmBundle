VfsmBundle
==========

## Installation

### Composer
 
Add the following dependencies to your projects composer.json file:

```json
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/kaywalker/VfsmBundle.git"
        }
    ],
    "require": {
        "hn/vfsm-bundle": "dev-master"
    }
```
